# Staff Guide

Last updated: 2026-04-02
Owner: Product Operations

## Login

1. Sign in with staff email and password.
2. If access is denied and message says account disabled, contact admin.

## Daily tasks

- Add inventory batches.
- Post stock movements.
- View and acknowledge expiry alerts.
- Review reorder and analytics data.
- Create audits, add lines, submit audits.

## What staff cannot do

- Create/edit drugs.
- Run forecast jobs.
- Access report export workflows.
- Manage staff accounts.
