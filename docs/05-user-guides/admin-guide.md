# Admin Guide

Last updated: 2026-04-02
Owner: Product Operations

## Login

1. Open client app in browser.
2. Enter admin credentials.
3. Confirm Dashboard and Staff tabs are visible.

## Daily operational flow

1. Review dashboard KPIs.
2. Trigger expiry scan and analytics refresh as needed.
3. Review alerts and acknowledge resolved ones.
4. Review reorder suggestions.
5. Process audits and reconcile submitted counts.
6. Queue and process required reports.

## Staff management

1. Open `Staff` tab.
2. Create account with temporary password.
3. Use `Reset Temp Password` when needed.
4. Use `Disable Account` for access revocation.
5. Use `Enable Account` to reinstate access.

## Inventory import

1. Open `Inventory` tab.
2. Download CSV template.
3. Fill required columns.
4. Upload CSV and verify success toast.
