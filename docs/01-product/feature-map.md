# Feature Map

Last updated: 2026-04-02
Owner: Product

## Dashboard

- KPI cards: stock value, expiring value, low stock count, critical alerts, forecast signals.
- Admin actions: run forecast, run expiry scan, refresh analytics.
- Forecast output table.

## Inventory

- Create drug (admin).
- Add inventory batch.
- Post stock movement (sale, return, adjustment).
- CSV import for inventory batches.
- Download CSV template + inline example rows.
- Search, sort, pagination, sticky table headers.

## Alerts and analytics

- View open expiry alerts.
- Acknowledge alert.
- View reorder suggestions.
- View consumption analytics.

## Audits

- Create stock audit.
- Add or update audit lines.
- Submit audit.
- Reconcile audit (admin approve/reject).

## Reports

- Queue report export.
- Trigger report processor.
- Track recent report jobs.

## Staff administration

- Create staff account.
- Reset temporary password.
- Disable account.
- Enable account.
